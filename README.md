# Bufsy

**Bufsy** — это кроссплатформенное CLI-приложение на Rust для синхронизации буфера обмена между устройствами по сети.

## О проекте

Bufsy позволяет быстро и безопасно передавать текстовые данные между компьютерами, автоматически копируя полученный текст в системный буфер обмена. Приложение поддерживает работу в локальной сети, использует современное шифрование для защиты данных и не требует сложной настройки.

## Основные возможности

- Синхронизация буфера обмена между устройствами по сети
- Приём и отправка текстовых данных через простой сервер
- Автоматическое копирование полученного текста в буфер обмена
- Шифрование и дешифрование сообщений с помощью AES-GCM
- Поддержка Wayland и X11 (через wl-clipboard-rs)
- Простое управление через командную строку

## Установка

1. Убедитесь, что у вас установлен [Rust](https://www.rust-lang.org/).
2. Клонируйте репозиторий:
    ```bash
    git clone <URL-репозитория>
    cd bufsy
    ```
3. Соберите проект:
    ```bash
    cargo build --release
    ```
4. Исполняемый файл будет находиться в `target/release/bufsy`.

## Безопасность

Bufsy использует современное шифрование (AES-GCM) для защиты передаваемых данных. Рекомендуется использовать уникальные ключи для каждого устройства и не передавать их по незащищённым каналам.
