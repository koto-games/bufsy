# Bufsy

Bufsy - это простое приложение командной строки, написанное на Rust, которое позволяет получать текст по сети и копировать его в буфер обмена.

## Возможности

- Запускает сервер для приема текста.
- Копирует полученный текст в системный буфер обмена.
- Демонстрирует шифрование и дешифрование сообщений с использованием AES-GCM.

## Использование

### `run`

Запускает сервер, который прослушивает входящие текстовые данные.

```bash
bufsy run [IP] [PORT]
```

- `IP`: IP-адрес для прослушивания (по умолчанию - локальный IP-адрес).
- `PORT`: Порт для прослушивания (по умолчанию - 9296).

При получении текста он будет скопирован в буфер обмена.

### `move`

Эта команда демонстрирует возможности шифрования и дешифрования.

```bash
bufsy move
```

## Сборка из исходного кода

1.  Клонируйте репозиторий:
    ```bash
    git clone <URL-репозитория>
    cd bufsy
    ```

2.  Соберите проект:
    ```bash
    cargo build --release
    ```

3.  Исполняемый файл будет находиться в `target/release/bufsy`.

## Зависимости

- `tokio` для асинхронной среды выполнения.
- `clap` для разбора аргументов командной строки.
- `axum` для веб-сервера.
- `aes-gcm` для шифрования.
- `wl-clipboard-rs` для работы с буфером обмена в Wayland.
- `anyhow` для обработки ошибок.
